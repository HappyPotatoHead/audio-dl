All notable changes to this project will be documented in this file.

## [1.0.3] - 2025-01-13
- Minor compatibility fixes
    - Fixed `sed` line when updating config

## [1.0.2] - 2025-01-12
- Minor compatibility fixes
    - Fixed csv parsing
    - Fixed `sed -i` for macOS compatibility

## [1.0.1] - 2025-01-12
- Minor bug fixes
    - Fixed dry run
    - Fixed batch download

## [1.0.0] - 2025-01-12

### Added
- Initial release
- YouTube audio download with artist/album organisation
- Batch download from text file
- Multiple format support (mp3, opus, flac, m4a, aac)
- Configuration management
- Verbose, quiet, and dry-run modes
- Cross-platform support (Linux, macOS)

### Dependencies
- yt-dlp
- ffmpeg (recommended)
